# Quechua Morphology Reference (Cuzco-Collao base, with Chanka/Ayacucho notes)

Source: Cahuana Q., Ricardo. *Manual de Gramática Quechua Cusco-Collao* (QHESWA, Sicuani-Peru, 2007), 48 pp.
Use case: RAG context for grounding Spanish→Quechua Chanka (`quy_Latn`) translation reasoning. Most morphology is shared across Southern Quechua; orthographic and a few lexical differences noted at the end.

Conventions: forms are written in the Cuzco-Collao orthography as in the PDF (with ejective `'` and aspirate `h`). For Chanka output, strip ejectives/aspirates (see §11).

---

## 1. Person / Number markers on verbs

Present-tense indicative verbal endings (PDF p.38). These are the canonical subject-agreement suffixes; the same person slots reappear in other tenses.

| Person | Suffix | Example (`llank'a-` "trabajar") | Gloss |
|---|---|---|---|
| 1sg | `-ni` | `llank'ani` | yo trabajo |
| 2sg | `-nki` | `llank'anki` | tú trabajas |
| 3sg | `-n` | `llank'an` | él/ella trabaja |
| 1pl incl | `-nchis` | `llank'anchis` | nosotros (incl.) trabajamos |
| 1pl excl | `-yku` | `llank'ayku` | nosotros (excl.) trabajamos |
| 2pl | `-nkichis` | `llank'ankichis` | ustedes trabajan |
| 3pl | `-nku` | `llank'anku` | ellos/ellas trabajan |

Notes (pp.16, 22): pronouns `ñoqa, qan, pay`; plurals `ñoqanchis` (incl.), `ñoqayku` (excl.), `qankuna`, `paykuna`. Verb endings agree with the pronoun and the pronoun is often omitted.

Chanka equivalents: `-niku` is the common Chanka allomorph for 1pl excl on consonant-final stems (cf. Cuzco `-yku`); `-nchik` is written with final `k` in Chanka (Cuzco `-nchis`).

## 2. Tense / Aspect / Mood

### 2.1 Present indicative
See §1. The bare person endings (`-ni, -nki, -n, -nchis, -yku, -nkichis, -nku`) encode present (p.38).

### 2.2 Past perfect (pretérito) `-rqa-` (p.37)
Concrete, completed action. Slot: stem + `-rqa-` + person.

| Form | Gloss |
|---|---|
| `llank'a-rqa-ni` | trabajé |
| `yanapa-rqa-ni` | ayudé |
| `saqe-wa-rqa-n-mi` | me dejó (with 1obj `-wa-` and direct evidential `-mi`) |
| `ranti-rqa-ni` | compré |

### 2.3 Past reportative / pluperfect `-sqa-` (p.37–38)
Used for actions the speaker did not witness directly: narration, stories, dreams, infancy memories. Often translated "dice que…", "habían…".

| Form | Gloss |
|---|---|
| `hampu-sqa` | dice que (él) vino |
| `ka-sqa` | dice que había |
| `q'otu-sqa` | dice que engañó |
| `wañuka-pu-sqa` | dice que murió |
| `urma-sqa-ni` | dice que me caí |

### 2.4 Past habitual `-q` + auxiliary `ka-` (p.38)
Stem + agentive `-q` + conjugated `kani/kanki/kan…` ("used to V").

| Form | Gloss |
|---|---|
| `oveja michi-q kani` | solía pastorear la oveja |
| `pelota-ta hayt'a-q kani` | solía jugar fútbol |
| `allinta llank'a-q kani` | solía trabajar bien |

### 2.5 Future (p.39)
Mixed paradigm by person:

| Person | Suffix | Example | Gloss |
|---|---|---|---|
| 1sg | `-saq` | `hamawt'a kasaq` | seré profesor |
| 2sg | `-nki` (= present form, disambiguated by time adverb `paqarin` etc.) | `rantipu-wa-nki` | me comprarás |
| 3sg | `-nqa` | `awayta yachachi-nqa` | enseñará a tejer |
| 1pl incl | `-sun` / `-sunchis` | `rantisun` | compraremos |
| 1pl excl | `-saqku` | — | |
| 2pl | `-nkichis` | — | |
| 3pl | `-nqaku` | `paqarin toca-nqa-ku` | mañana tocarán |

Future subordinate / second-person object: `-sqayki` ("I will V you", p.39) — `yachachi-sqayki` = te enseñaré; `rantipu-sqayki` = te compraré.

### 2.6 Imperative (p.37)
Bare stem + `-y` for 2sg, `-ychis` for 2pl, or special forms.

| Form | Gloss |
|---|---|
| `phaway` | corre / anda |
| `uyaykita maqchikuy` | lávate la cara |
| `usqhayta ripuy` | vete rápido |
| `qoymuy` | dale / tráelo |
| `hamuy` | ven |
| `hamu-ychis` | vengan |

### 2.7 Prohibitive
`ama ...-chu` (pp.31, 37): `ama-n machanki-chu` = "no debes emborracharte"; `aman rimankichu` = "no debes hablar". `ama-puni` = "favor que no"; `ama-raq` = "todavía no".

### 2.8 Subjunctive / Conditional (p.36)
Expressed with `-pti-` (different-subject) or `-spa` (same-subject) plus matrix verb, or with conjunction `chayqa` "si":

| Form | Gloss |
|---|---|
| `chaypi ka-q-tiyki hamusaq` | si estás allí vendré |
| `apa-mu-q-tiyki-qa sirasaqmi` | si traes coseré |
| `sumaq ka-q-tin-qa rantisun` | si está bueno, compraremos |
| `hamunki chayqa qosqayki` | si vienes te daré |
| `munaqtiykiqa sut'inchasaqmi` | si tú quieres, aclararé |

### 2.9 Obligative (p.36–37)
`-na` + possessive + `kay` ("hay que / tengo que"):
`ñoqa-qa chaypi ka-na-y-puni-n` = yo tengo que estar allí; `llaqtataqa ri-na-yki-puni` = de todas maneras tienes que ir al pueblo.

### 2.10 Infinitive `-y` (p.36)
Bare nominalized form: `pukllay` jugar, `qhaway` mirar, `rantiy` comprar, `apay` llevar, `siray` coser, `yachachiy` enseñar, `phaway` correr, `qhatuy` vender.

## 3. Case suffixes on nouns

All examples from pp.16–19.

| Case | Suffix | Function | Example | Gloss |
|---|---|---|---|---|
| Accusative | `-ta` | direct object; sometimes indirect | `manka-ta apamuy` | trae la olla |
| | | | `t'anta-ta rantimuy` | cómpralo el pan |
| | | | `wawqey-ta-n yanapamusaq` | ayudaré a mi hermano |
| Illative / Directional | `-man` | "a, hacia" (motion goal, indirect obj) | `tayta-yki-man-si willanki` | dice que avises a tu papá |
| | | | `chay-man rishan` | a dónde va este camino |
| | | | `ch'isi-man-ñas ripunqa` | dice que en la noche se va |
| Ablative | `-manta` | origin, separation, material | `kunan-manta-qa ama-n rimankichu` | no debes hablar desde ahora |
| | | | `trigo-manta t'anta-qa ruwakun` | el pan se hace de trigo |
| | | | `Layo-manta-n hamushani` | estoy viniendo de Layo |
| Locative | `-pi` | location, time, instrument-of-transport | `may-pi-n rumi kashan` | dónde está la piedra |
| | | | `iskay killa-pi yachaqashaq` | aprenderé en 2 meses |
| | | | `caballo-pi-n tiyuyki hamusqa` | tu tío vino a caballo |
| | | | `wasi-pi-ña-n samasaq` | descansaré ya en la casa |
| Instrumental / Comitative | `-wan` (after C, also `-piwan`) | accompaniment, instrument | `qan-wan-mi rimayta munani` | quiero hablar contigo |
| | | | `toro-wan-mi llank'asaq` | trabajaré con la yunta |
| | | | `Pablo-piwan-mi hamusqa` | había venido con Pablo |
| | | | `ñoqa-piwan-mi risun` | iremos conmigo |
| Dative / Benefactive | `-paq` | "para", purpose, benefit/harm | `mama-yki-paq-mi kay qolqe-qa` | este dinero es para tu mamá |
| | | | `muhu-paq-chu kay papa` | ¿esta papa es para semilla? |
| Genitive | `-q` (after V) / `-pa` (after C) | possessor | `Rosa-q lliklla-n-mi kay-qa` | Esta es la manta de Rosa |
| | | | `Jorge-q-mi kay ch'ulluqa` | este chullo es de Jorge |
| | | | `añas-pa t'oqo-n-mi` | esa es cueva de zorrino |
| | | | `Juan-pa p'uku-n` | pocillo de Juan |
| | | | `pi-qpa-taq kay waka-ri` | ¿de quién es esta vaca? (both suffixes) |
| Limitative / Terminative | `-kama` | "hasta" (distance, time, equality) | `Qosqo-kama-n Julianqa rishan` | Julián viaja hasta Cusco |
| | | | `wata-kama suyasqayki` | te espero hasta un año |
| | | | `kutimuna-y-kama` | hasta que regrese |
| | | | `kasqan-kama kay qowikuna` | estos cuyes son iguales |
| Causal | `-rayku` | reason, cause "por" | `qan-rayku-n muchurqan` | sufrió por ti |
| | | | `waway-rayku-n llank'ani` | trabajo por mi hijo |
| | | | `ima-rayku-n mana mikhunkichu` | ¿por qué no comes? |

The manual does not give a dedicated `-nta` "route/through" section; in Chanka it is the standard "by way of, through" suffix and behaves like the other case markers.

## 4. Evidentials and validators

The manual treats these as enclitic affirmation/doubt particles rather than a single paradigm; they attach after case/person endings and mark the speaker's source.

| Marker | Function | PDF notes / example |
|---|---|---|
| `-mi` / `-n` | Direct evidential / assertion: speaker witnessed or vouches for. After consonant → `-mi`, after vowel → `-n`. | pp.13, 23–24, 31, 33: `wasiyoq-mi`, `pi-n chay runa-ri`, `ñoqa yunkamanta-n ka-ni`, `chay-mi hamuni` "vine por eso". |
| `-si` / `-s` | Reportative: hearsay, "dicen que". | pp.16, 37–38: `tayta-yki-man-si willanki`, `taytay-si …hampusqa`, `atoq-si llaqta-y-pi ka-sqa`. |
| `-chá` | Inferential / conjecture "ha de ser, quizá". | pp.13, 16, 31: `yaqapas-chá`, `erqey-chá` = debe ser mi hijo; `qowi-ta-chá munashan` = seguro quiere cuy; `hina-chá` posiblemente; `imaynachá` cómo será. |
| `-puni` | Emphatic / definite "de veras, por seguro". | pp.13, 24, 30–31: `wasiyoq-puni` legítimo dueño; `kiki-y-puni-n rikuni` = lo vi yo mismo; `manapuni` de ninguna manera; `amapuni` favor que no. |
| `-yá` | Polite emphasis / "por favor". | p.13: `hinata-yá` = por favor. |
| `-pas` / `-pis` | Indefinite "también, cualquiera". | pp.16, 24: `Luista-pas waqyamunin` = también llamé a Luis; `pi-pas llank'achun` = alguien que trabaje. |
| `-raq` | "todavía, aún". | p.31: `mana-raq` todavía no. |
| `-taq` | Contrastive coordinator / wh-emphatic. | pp.31, 33: `pay-taq samanqa` "y él descansará"; `chay-taq mana hamuwaqchu` "cuidado que no vengas"; in questions: `pi-taq?`, `hayk'aq-taq?`. |
| `-ri` | Coordinator question particle "¿y…?". | p.32: `ñoqa-ri hamusaqchu` = ¿y yo vendré?; `Benancio-ri ripunchu` = ¿y Benancio se fue? |
| `-chu` | Yes/no question; also negation marker. | pp.16, 24, 31, 32: `hamunki-chu` = ¿vienes?; `manan pi-pas hamu-n-chu` = nadie ha venido. |

Stacking order (from examples): root – derivation – tense – person – case – number/topic – evidential. E.g. `ñañayki-man-mi saqesqa` = "había dejado a tu hermana" (DAT + direct evid.).

## 5. Possession / Nominal person

Possessive suffixes attach directly to the noun (pp.21, 23, 26). Pronouns plus genitive (`ñoqa-q, qan-pa, pay-pa`) reinforce them.

| Person | Suffix | Example (`alqo` "perro") | Gloss |
|---|---|---|---|
| 1sg | `-y` | `alqo-y` | mi perro |
| 2sg | `-yki` | `alqo-yki` | tu perro |
| 3sg | `-n` (after V) / `-nin` (after C) | `alqo-n` | su perro |
| 1pl incl | `-nchis` | `alqo-nchis` | nuestro perro (incl.) |
| 1pl excl | `-yku` (Cuzco) / `-niku` (Chanka) | `alqo-yku` | nuestro perro (excl.) |
| 2pl | `-ykichis` | `alqo-ykichis` | perro de ustedes |
| 3pl | `-nku` | `alqo-nku` | perro de ellos/ellas |

Genitive-marked pronoun forms (p.23):
- 1sg `ñoqa-q`, 2sg `qan-pa`, 3sg `pay-pa`
- 1pl excl `ñoqayku-q`, 1pl incl `ñoqanchis-pa`, 2pl `qankuna-q`, 3pl `paykuna-q`.

Note: with a possessor noun, both genitive on possessor and possessive on possessed surface together: `Rosa-q lliklla-n-mi` "la manta de Rosa".

## 6. Nominal plural, distributive, augmentative, diminutive (pp.16, 20–22)

| Suffix | Function | Example | Gloss |
|---|---|---|---|
| `-kuna` | Plural of nouns / pronouns | `punku-kuna`, `mallki-kuna`, `paqocha-kuna`, `pay-kuna` | puertas, árboles, alpacas, ellos |
| `-yku` | Exclusive 1pl plural marker on pronoun | `ñoqa-yku` | nosotros (excl.) |
| `-nchis` | Inclusive 1pl plural marker on pronoun | `ñoqa-nchis` | nosotros (incl.) |
| `-sapa` | Augmentative ("…ón, abundante en") | `uma-sapa`, `wiksa-sapa`, `maki-sapa` | cabezón, barrigón, brazón |
| `-cha` | Diminutive / affective | `warmi-cha`, `alqo-cha`, `t'ika-cha` | mujercita, perrito, florcita |
| reduplication | Collective/abundance | `mallki-mallki`, `rumi-rumi`, `sara-sara` | bosque, pedregal, maizal |
| `-pura` | Distributive "entre" | `qhari-pura`, `qankuna-pura` | entre varones, entre ustedes |
| `-masi` | Associative "co-, fellow-" | `runa-masi`, `wasi-masi`, `llank'aq-masi` | prójimo, vecino, compañero de trabajo |
| `-ntin` | Inclusive "together with" (numerals) | `iskay-ni-nchis`, `kinsa-nti-nchis` | nosotros dos, nosotros tres |
| `-kacha`, `-ykacha`, `-paya` | Frequentative / iterative on verbs | `phawa-kachay`, `puri-ykachay`, `waqa-payay` | corretear, andar de un lugar a otro, llorar repetidas veces |

Connector `-ni-` / `-nti-` is inserted between consonant-final base and vowel-initial suffix (`huk-ni-nchis` "uno de nosotros", `tawa-nti-nchis` "los cuatro de nosotros").

## 7. Verbal derivational suffixes

Most appear scattered across pp.20–21, 34–38. Slot order: ROOT – (valency) – (aspect) – tense – person.

| Suffix | Function | Example | Gloss |
|---|---|---|---|
| `-chi` | Causative "hacer V" | `yacha-chi-y`, `yacha-chi-saq` | enseñar (hacer saber), enseñaré |
| `-ku` | Reflexive / middle / for-oneself | `maqchi-ku-y`, `wañu-ka-pu-sqa` | lavarse, se murió |
| `-na` | Obligative / nominalizer of place-instrument | `hampi-na wasi`, `qora-na`, `t'oqo-na`, `ka-na-y` | hospital, azadón, perforador, (tener que) ser |
| `-y` | Infinitive / action nominal | `pukllay, rantiy, apay, siray` | jugar, comprar, llevar, coser |
| `-q` | Agentive nominalizer "el que V"; also past habitual w/ `ka-` | `llank'a-q-masi`, `michi-q kani` | compañero de trabajo, solía pastorear |
| `-sqa` | Past participle / reportative past | `sayk'u-sqa kashanki`, `kasha-sqa-ni` | estás cansado, dice que estuve |
| `-na` | Future participle / "lo que ha de V" | `kuti-mu-na-y-kama` | hasta que yo regrese |
| `-spa` | Same-subject adverbial gerund | `ka-spa`, `wawa ka-spa-s urmasqani` | siendo, siendo bebé me caí |
| `-pti-` (+ poss.) | Different-subject adverbial "cuando / si" | `chaypi ka-q-tiyki`, `apa-mu-q-tiyki-qa` | si estás allí, si traes |
| `-sha-` / `-chka-` | Progressive "estar V-ndo" | `hamu-sha-n`, `ri-sha-n`, `mikhu-sha-n`, `para chaya-sha-n` | viene/está viniendo, está yendo, está comiendo, está lloviendo |
| `-mu` | Cislocative (motion toward speaker / "venir a V") | `hamu-y`, `apa-mu-y`, `willa-mu-wan`, `kuti-mu-na-y` | venir, traer, vino a avisarme, mi regreso |
| `-pu` | Benefactive / completive ("for someone / definitively") | `wañu-ka-pu-sqa`, `ripu-nqa`, `qopu-way` | se murió, se irá, dámelo |
| `-wa-` | 1st person object "me / a mí" | `saqe-wa-rqa-n-mi`, `willa-wa-n`, `qhawa-chi-wa-n` | me dejó, me dice, me muestra |
| `-naku` | Reciprocal "se entre sí" | `huñu-naku-y`, `yanapa-naku-y` (cf. p.31 `yanapa-naku-sun`) | reunirse, ayudarse |
| `-cha-`, `-ya-`, `-na-` | Verbalizers from nouns/adjectives | `wasi-cha-y`, `wira-ya-y`, `rumi-ya-y`, `k'uchu-na-y`, `sonqo-na-y` | techar casa, engordar, endurecerse, arrinconar, poner dentro |
| `-kacha-`, `-ykacha-`, `-paya-` | Frequentative | (see §6) | corretear, etc. |

Sentence-level negation: `mana(n) … -chu` (pp.24, 31, 38). The validator `-n/-mi` attaches to `mana` in declarative negatives: `mana-n pi-pas hamu-n-chu` "nadie ha venido"; `mana-n mayqen-pas ka-n-chu` "ninguno de ellos está".

## 8. Negation and prohibition (pp.24, 31, 37)

| Pattern | Meaning | Example |
|---|---|---|
| `manan … -chu` | declarative negation | `mana-n pi-pas muna-n-chu` = "nadie lo quiere" |
| `mana` alone | "no" (short answer / pre-verbal) | `mana ari` = no pues; `mana-káw` = no quiero |
| `mana-raq` | "todavía no" | `mana-raq` |
| `mana-puni` | "de ninguna manera" | — |
| `ama … -chu` | prohibitive (2nd-person imperative neg.) | `ama machanki-chu` = no te emborraches; `ama saqe-y-chu` = no lo dejes |
| `ama-raq` | "todavía no debe" | — |
| `ama-puni` | "favor que no" | — |
| `ama-pas` | "quizá no" | — |

Indefinite "nadie / nada / ninguno" is built as `manan + WH-pas … -chu`: `manan pipas hamunchu`, `manan mayqenpas kanchu`, `manan mayqenpas wirachu` (p.24).

## 9. Question particles and wh-words

### 9.1 Yes/no questions (pp.16, 24, 32)
Attach `-chu` to the focused element: `hamunki-chu mana-chu` = "¿vienes o no vienes?"; `risaq-chu ama-chu` = "¿voy o no voy?".

### 9.2 Wh-words (pp.24, 30, 43)
| Word | Gloss |
|---|---|
| `pi` | quién |
| `ima` | qué |
| `mayqen` | cuál |
| `may` | dónde |
| `hayk'aq` | cuándo |
| `hayk'a` | cuánto |
| `imayna` | cómo |
| `ima-rayku` | por qué |

Wh-emphatic with `-taq`: `pi-taq chay runa-ri` "¿quién es esa persona?"; `mayqen-tan munanki` "¿a cuál de ellos quieres?"; `hayk'aq-taq Qosqomanta kutimunki?` "¿cuándo vuelves de Cusco?" (p.43).

### 9.3 Coordinator/question `-ri` (p.32)
"¿y …?": `ñoqa-ri hamusaqchu` "¿y yo vendré?"; `Benancio-ri ripunchu` "¿y Benancio se fue?".

### 9.4 Indefinites from wh + `-pas` (p.24)
`pi-pas` alguien, `mayqen-pas` cualquiera, `ima-pas` cualquier cosa; with `mana(n) … -chu` → "nadie / ninguno / nada".

## 10. Word order in Southern Quechua

The manual exemplifies sentences as **S (O) V** with modifiers preceding heads (pp.40–43).

Examples directly from p.42–43, with marked subject (s) and predicate (p):
- `wawqekuna paqarin tocanqaku` — "mañana los hermanos van a tocar". S–Adv–V.
- `Antonio-qa waka-ta wata-shan` — "Antonio está amarrando a la vaca". S–O–V.
- `khiska-n chaki-y-ta t'urpurqo-wan` — "la espina pinchó mi pie". S–O–V (with 1obj `-wa-`).
- `taki-q-kuna llaqta-pi paqarin toca-nqa-ku` — "el conjunto musical tocará mañana en el pueblo". S–LOC–TIME–V.
- `Maria mama-qa p'acha-ta-n qhatu-shan plaza tienda-pi` — "la señora María está vendiendo ropa en la tienda de la plaza". S–O–V–LOC (locative may follow V).
- `sara kallcha-y-man-mi ri-nqa-ku Anselmo-wan Juan-wan` — "Anselmo con Juan irán a la cosecha de maíz". V–S, demonstrating that conjoined subjects can follow the verb when topic-marked.

Other order tendencies visible in the manual:
- **Modifier before head**: `orqo asno` "el burro" (masc + N, p.15), `china alqo` "la perra"; `kay waskha-qa` "esta soga"; `hatun poqoy` "grande maduración = febrero".
- **Possessor (GEN) before possessed (POSS)**: `Rosa-q lliklla-n` (p.18), `wawqey-pa chakra-n-pi` "en la chacra de mi hermano" (p.40).
- **Postpositional case marking**: case suffixes attach to the noun (never preposition).
- **Negation `mana(n)` and prohibitive `ama` precede the verb**; closing `-chu` on the verb (or focused element).
- **Subordinate clauses precede main verb** via `-spa` / `-pti-` participles: `chaypi kaqtiyki hamusaq` "si estás allí, vendré".

## 11. Chanka (Ayacucho `quy_Latn`) deviations from Cuzco-Collao

Most of the morphology above transfers directly. The systematic differences a translator must apply are:

### 11.1 Orthography / phonology
Chanka uses only **plain stops**; Cuzco contrasts plain ~ aspirated ~ ejective. When porting Cuzco lemmas to Chanka, drop the aspiration and the ejective:

| Cuzco | Chanka | Gloss |
|---|---|---|
| `p', ph` | `p` | `p'unchaw` → `punchaw` "día" |
| `t', th` | `t` | `t'anta` → `tanta` "pan" |
| `ch', chh` | `ch` | `ch'aki` → `chaki` "seco/pie" |
| `k', kh` | `k` | `k'anka` → `kanka` "gallo" |
| `q', qh` | `q` | `qhepa` → `qipa` "atrás"; `q'omer` → `qumir` "verde" |

Vowels: Chanka standard orthography (SIL/MINEDU) uses **3 vowels `a, i, u`**; Cuzco writes `e, o` allophones next to `q`. So Cuzco `qosqo`, `qolqe`, `q'omer` → Chanka `qusqu`, `qullqi`, `qumir`. The manual itself argues for 5-vowel writing (p.10); for Chanka output prefer 3-vowel.

`h` is preserved in both; `sh` is rare in Chanka — the progressive is `-chka-` (see below).

### 11.2 Inflectional differences

| Slot | Cuzco | Chanka | Notes |
|---|---|---|---|
| Progressive | `-sha-` (`hamushan`) | `-chka-` (`hamuchkan`) | Functionally identical "está V-ndo". The Cuzco manual uses `-sha-` throughout pp.10–11, 38. |
| 1pl incl | `-nchis` | `-nchik` | Final `s` → `k` in Chanka standard. |
| 1pl excl | `-yku` | `-yku` after vowel; `-niku` after consonant | Both varieties allow `-niku`; Chanka writes it more often. |
| 2pl | `-ykichis` | `-ykichik` | Same final `s` → `k`. |
| Genitive | `-q` / `-pa` | `-pa` (after both V and C; `-p` also seen) | Chanka generalises `-pa`; `Rosaq` → `Rosapa`. |
| Ablative | `-manta` | `-manta` | identical |
| Past reportative | `-sqa` | `-sqa` | identical |
| Past direct | `-rqa` | `-rqa` (also `-ra-` in fast speech) | identical in writing |

### 11.4 Quick conversion rules for the translator
1. Strip all `'` and lowercase `h` after a stop letter when porting a Cuzco lemma.
2. Map `e→i` and `o→u` (most positions; safe near `q`).
3. Replace progressive `-sha-` with `-chka-`.
4. Replace `nchis / ykichis` with `nchik / ykichik`.
5. Use `-pa` for genitive after both vowels and consonants.
6. Keep all case suffixes (`-ta, -pi, -man, -manta, -wan, -paq, -kama, -rayku, -nta`) and validators (`-mi/-n, -si/-s, -chá, -puni, -pas, -taq, -ri, -chu`) exactly as in Cuzco.
7. Sentence template: `S [Adv/Loc/Time] [O-ta] V-(deriv)-(asp)-(tense)-(person)-(case)-(validator)`.

## 12. Quick reference: day / month / season vocabulary (p.45)

Useful for date-bearing translations.

- Days: `Killachaw` lunes, `Atichaw` martes, `Qoyllurchaw` miércoles, `Illapachaw` jueves, `Ch'askachaw` viernes, `K'uychichaw` sábado, `Apuchaw`/`Intichaw` domingo.
- Months: `Qholla poqoy` enero, `Hatun poqoy` febrero, `Pawkarwaray` marzo, `Ariway` abril, `Aymuray` mayo, `Intiraymi` junio, `Antasituwa` julio, `Qhapaqsituwa` agosto, `Umaraymi` septiembre, `Kantaray` octubre, `Aya marka` noviembre, `Poqoyraymi` diciembre.
- Seasons (`mita`): `Tarpuy mita` primavera, `Paray mita` verano, `Cosecha mita` otoño, `Qasa/chiri mita` invierno.
